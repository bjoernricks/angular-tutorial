Es ist über .$watch möglich Variablen zu überwachen

Asynchrone Veränderung der Variablen ohne spezielle AngularJS Hilfsmittel werden nicht registriert. D.h. die Integration von anderen JavaScript Bibliotheken ist meist ohne speziellen AngularJS Code nicht möglich. Verändert eine andere JavaScript Bibliothek Werte im DOM, muss dies AngularJS erst bekannt gemacht werden. Hierzu dient die .$apply() Funktion. Siehe Angular Scope Dokumentation.
