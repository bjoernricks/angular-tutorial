Was ist neu?
  • Über die scope Property können Daten an den isolierten Scope der Direktive übergeben werden

Die Syntax für das Übergeben von Daten an den isolierten Scope der Direktive lautet variable: '=attr'. Dabei ist variable der Name der Property im neuen isolierten Scope und attr der Name des HTML Attributes im Template. Wird attr weg gelassen und nur variable: '=' verwendet, lautet das Attribute im HTML Code ebenfalls variable. Beispiel: Für

würde scope wie folgt lauten:
scope: { myAttr: '=', }
