Angular JS beinhaltet bereits von Haus aus eine Komponente, mit der Single-Page-Applications realisierbar sind. Über den Service ngRoute lassen sich ausgewählte URLs mit View und Controllern verbinden und so eine Navigation in der Anwendung realisieren. Da ngRoute nur flache Navigationshirachien umsetzen kann, ist im Angular UI Projekt eine weitere Router Komponenten der Angular UI Router entstanden. Dieser Router definiert ein zustandsbasiertes Verhalten, das eine Hirarchie von Controllern und Views verwenden kann.
