Was ist neu?
  • Für jeden Aufruf der Direktive wird ein eigener isolierter Scope erstellt
  • Die Variable ctrl ist somit im Scope nicht mehr verfügbar

Wird für die scope Property im Directive defintion object ein neues Objekt erstellt, wird für jeden Aufruf der Direktive ein isolierter Scope erzeugt. Dieser Scope ist somit leer.

Da hier im Controller der Direktive auf eine Property der nicht existierenden Variable ctrl des Scopes zugegriffen wird, kommt es zu einer Exception und die Ausführung des Codes wird gestoppt.
