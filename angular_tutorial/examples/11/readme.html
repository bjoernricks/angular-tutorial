Was ist neu?
  • Dependency Injection ist auch mit Minification verwendbar

Minification ist eini bei JavaScript häufig eingesetztes Verfahren, um die zu übertragenden Datenmengen zu reduzieren. Dabei werden Präprozessoren auf den JavaScript (und CSS) Quellcode angewandt, welche den Codeumfang reduzieren. Bei diesem Prozess werden z.B. Leerzeichen, Umbrüche, lange Variablennamen entfernt.

Da AngularJS über die Variablen erkennet, welche Objekte als Abhängigkeiten injected werden muss, müssen die Variablen auch im minifizierte Code erhalten bleiben. Da Zeichenketten nicht minifiziert werden können, werden diese einfach der Controller Funktion vorangestellt. Die Funktionsparameter können dann auch umbenannt werden und müssen nicht mehr wie hier $scope und $timeout heißen.
