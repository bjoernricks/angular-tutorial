  • Services können Daten und Methoden kapseln

Unser NameService verwendet den $http Service, um die Liste von Namen bereitzustellen. API

Der Angular JS $http Service kapselt den Zugriff auf externe HTTP Resourcen mittels XMLHttpRequest.

Die $http Methoden geben sogenannte Promises zurück. Promises haben eine then Methode, die eine Callback-Funktion mit einem optionalen Argument erwarten. Im Fall von $http wird dabei ein response Objekt übergeben. Das Callback wird erst aufgerufen, wenn die Daten vollständig geladen wurden.

Die vollstänfige Signatur der then Methode des Angular JS Promise lautet: then(successCallback, errorCallback, notifyCallback).

Dem Controller ist das Backend für die Namensdaten vollkommen egal.
