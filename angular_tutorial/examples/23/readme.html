Was ist neu?
  • Angular JS $http Service wird benutzt
  • Services können Daten und Methoden kapseln

Unser NameService verwendet den $http Service, um die Liste von Namen bereitzustellen.

Der Angular JS $http Service kapselt den Zugriff auf externe HTTP Resourcen mittels XMLHttpRequest.

Die $http Methoden geben sogenannte Promises zurück. Promises haben eine then Methode, die ein Callback mit einem optionalen Argument erwarten. Im Fall von $http wird dabei ein response Objekt übergeben. Das Callback wird erst aufgerufen, wenn die Daten vollständig geladen wurden.

Dem Controller ist das Backend für die Namensdaten vollkommen egal.
