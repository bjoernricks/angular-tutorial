Was ist neu?
  • .filter statt .controller
  • Filter gibt eine Funktion zurück

Über .filter können Filter in Angular JS registriert werden.

Ein Filter gibt immer eine Filterfunktion zurück. In dieser Filterfunktion können die Eingabewerte manipuliert werden.

Auch Filtern können Abhängigkeiten injected werden:

truncateFilter.$inject = ['$log']; function truncateFilter($log) { function truncate(value) { var max = 20; var out = value; if (out.length > max) { $log.debug('Truncating ' + out); out = out.substr(0, max) + ' ...'; } return out; } return truncate; }
