Was ist neu?
  • $timeout statt setTimeout
  • Dependency Injection

Die spezielle AngularJS Funktion ersetzt setTimeout. Intern ruft diese Funktion $scope.$apply() auf. S

$timeout ist ein AngularJS Service, der an die Controller Funktion übergeben wird. AngularJS verwendet hierzu das Dependency Injection Designmuster. Anhand der Variablennamen übergibt AngularJS die vorher registrierten Objekte an die jeweilige JavaScript Funktion.

Auch $scope wird per Dependency Injection an die Funktion übergeben. Es ist somit auch möglich einen Controller zu schreiben, der die $scope Variable gar nicht verwendet.
