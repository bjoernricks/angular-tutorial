Was ist neu?
  • Controller as Syntax im Template
  • Variablen werden im Controller this zugewiesen

Vorteile:

  • Klare Kapselung von Komponenten in Angular JS
  • Vorwärtskompatibilität zu Angular 2
Video: Refactoring to Components

Daten werden an einen Controller gebunden. Die Daten befinden sich somit nicht mehr direkt im ominösen $scope. Sie befinden sich immer noch indirekt im $scope. Die Controller as Syntax entspricht in etwa:

angular.module('nameApp', []) .controller('NameCtrl', NameCtrl); NameCtrl.$inject = ['$scope', '$log', '$timeout']; function NameCtrl($scope, $log, $timeout) { var ctrl = this; $scope.ctrl = ctrl; ctrl.firstName = 'John'; ctrl.lastName = 'Doe'; $timeout(function() { $log.debug('Changing lastname to Smith'); ctrl.lastName = 'Smith'; }, 1000); }
