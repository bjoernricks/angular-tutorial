AngularJS Examples

Inspired by Curran Kelleher

Angular JS ist ein JavaScript Framework, dass eine Model View Controller Architektur verwendet (häufig ist hier auch von Model View ViewModel die Rede).

  • Die View wird dabei durch HTML Templates mit angereichertem Markup repräsentiert. Das angereicherte Markup besteht dabei aus sogenannten Angular Expressions und Direktiven.
  • Der Controller (bzw. das ViewModel) wird in JavaScript implementiert und kapselt die Business-Logik. Ebenfalls regelt der Controller den Zugriff des Templates auf die im Model liegenden Daten.
  • Das Datenmodell wird in sogenannten Scopes abstrahiert.
